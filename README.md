# 🐰 Leka-Leka: Teman Belajar Tanpa Batas

**Leka-Leka** (diambil dari kata Dis-leksia) adalah aplikasi mobile yang dirancang khusus untuk mendukung anak-anak dengan disleksia. Menggunakan metode interaktif **VAKT (Visual, Auditory, Kinesthetic, Tactile)**, aplikasi ini membantu anak-anak mengatasi hambatan membaca dengan cara yang menyenangkan dan inklusif.

---

## Fitur Utama
### Screening Awal (Early Detection)
- **RAN Test (Rapid Automatized Naming):** Mengukur kecepatan pemrosesan visual anak menggunakan grid objek familiar.
- **Phonemic Check:** Menguji kemampuan anak dalam membedakan bunyi huruf yang mirip (P vs B, D vs T).

### Visual Learning
- **Syllable Highlighter:** Pemecahan kata menjadi suku kata berwarna untuk memudahkan pemetaan visual.
- **Adaptive Interface:** Penggunaan warna latar belakang krem (`#FFFBEB`) dan font khusus untuk mengurangi *visual stress*.

### Auditory Training
- **Audio Duel:** Game interaktif untuk mencocokkan bunyi dengan gambar yang tepat.
- **Voice Feedback:** Setiap teks dapat diklik untuk mendengarkan pengucapan yang benar.

### Kinesthetic & Tactile
- **Letter Tracing:** Latihan menulis huruf dengan mengikuti garis bantu untuk membangun *muscle memory*.
- **Word Scramble:** Menyusun kata dengan mekanisme *drag-and-drop* yang interaktif.

---

## 🛠️ Tech Stack

- **Framework:** React Native (Expo)
- **Navigation:** Expo Router (File-based routing)
- **Logic:** JavaScript (ES6+)
- **Assets:** Custom Vector Illustrations
